## 一、核心概念

**一句话**：在世界树上指挥昆虫大军，保护金蝉抵御天敌，攀登至树冠羽化成仙。

**类型**：竖版自动射击 + 昆虫养成  
**平台**：微信小游戏  
**美术风格**：自然手绘风（水彩 + 国风）  
**参考**：向僵尸开炮（Survivor.io）玩法 + 虫师/夏目友人帐美术氛围

## 二、核心玩法

### 2.1 战场布局（竖版）

```
┌─────────────────────┐
│  天敌从上方不断下压   │  ← 砖块区域（天敌）
│  🕷 🐛 🪳 🐜 🦂     │
│                     │
│                     │
│  🪲  🐝  🦋  🐜  🦟 │  ← 虫兵水平排列线
│         🪙          │  ← 金蝉（中央）
│                     │
└─────────────────────┘
```

- **金蝉**：屏幕中下方中央，不可移动（或小范围左右晃动）
- **虫兵**：在金蝉上方水平一条线排列，自动攻击
- **天敌**：从上方生成，缓慢下压逼近金蝉
- **金蝉无攻击能力**，只有血量，被天敌接触则扣血
- **Game Over**：金蝉血量归零

### 2.2 虫兵系统（原武器系统）

金蝉通过蝉鸣召唤昆虫兵种，每种虫兵有独立攻击方式和升级树。

| 虫兵 | 原型 | 攻击方式 | 定位 |
|------|------|----------|------|
| 🪲 屎壳郎 | 迫击炮 | 推粪球向上抛射→砸落爆炸AOE | 范围轰炸 |
| ⚡ 电蜂 | 闪电链 | 蜂群飞出→跳跃蜇刺连锁 | 多目标连锁 |
| 🦟 追踪蚊 | 导弹 | 锁定最强天敌→持续吸血 | 单体追踪 |
| 🕷 蜘蛛 | 陨石 | 从上方降落→织网覆盖AOE | 重型AOE |
| 🐜 蚂蚁军团 | 无人机 | 列阵排列→高频切割 | 持续DPS |
| 🦋 刃翅蝶 | 旋刃 | 弹墙飞舞→持续切割后排 | 后排清扫 |
| 🔥 火蚁 | 白磷弹 | 从天砸下→落地燃烧蔓延 | 区域DOT |
| 🪱 寄生虫 | 离子射线 | 锁定宿主→持续吸取生命 | 单体DPS之王 |

### 2.3 金蝉能力

- **蝉鸣**：被动，自动召唤/指挥虫兵
- **金蝉本体无攻击**，纯防御型核心
- **血量**：被天敌接触扣血，血量=生命值
- **蜕变能力**：远期设计，暂不实现

### 2.4 天敌系统（原砖块系统）

天敌从上方生成，缓慢下压，接触金蝉造成伤害。

#### 普通天敌（按世界树层级）

| 层级 | 天敌 | 特点 |
|------|------|------|
| 根系（1-10） | 蛆虫、蜈蚣、地虱 | 低血量，慢速 |
| 树干（11-25） | 蛀虫、白蚁、树蜂 | 中血量，成群 |
| 枝叶（26-50） | 毛毛虫、蚜虫、叶蜂 | 特殊能力（减速/护盾） |
| 花冠（51-75） | 胡蜂、花螳、寄生蜂 | 高血量+特殊能力 |
| 云端（76-100） | 雷虫、风蛾、天蚕 | Boss级小怪 |

#### Boss天敌（每层终点）

| Boss | 原型 | 特点 |
|------|------|------|
| 🦎 壁虎 | 冲锋者 | 快速冲刺，舌头远程攻击 |
| 🐸 树蛙 | 护盾卫士 | 膨胀护甲，毒液弹 |
| 🦂 蝎子 | 召唤师 | 召唤蝎子幼崽 |
| 🐍 青蛇 | 激光炮台 | 吐信远程毒射 |
| 🦅 食虫鸟 | 幽影刺客 | 俯冲突袭，高闪避 |

## 三、关卡系统

### 3.1 世界树分层

```
  ☁️ 云端·风暴（76-100）
  🌸 花冠·蜜域（51-75）
  🌿 枝叶·绿海（26-50）
  🪵 树干·裂谷（11-25）
  🌑 根系·暗域（1-10）
```

- 关卡选择界面 = 世界树纵向路径（已有六边形节点可复用）
- 每层Boss击杀 = 进入下一层
- 背景渐变：暗色潮湿 → 温暖阳光 → 花海 → 云端

### 3.2 单局流程

1. 选择关卡（世界树节点）
2. 进入战斗：天敌从上方下压
3. 升级：获得经验→三选一（新虫兵/虫兵强化/金蝉强化）
4. Boss战：每关末尾出现天敌Boss
5. 通关/失败

## 四、美术方向

### 4.1 风格关键词
- **水彩手绘** + **国风工笔虫草**
- 参考：齐白石虫草画 + 虫师动画
- 柔和色调，墨线勾边，水彩晕染
- 背景：宣纸质感 + 淡墨山水

### 4.2 配色方案

| 层级 | 主色调 |
|------|--------|
| 根系 | 深褐+暗绿+菌丝荧光 |
| 树干 | 暖棕+琥珀+树液金 |
| 枝叶 | 翠绿+阳光斑驳 |
| 花冠 | 粉紫+蜜金+花瓣飘落 |
| 云端 | 靛蓝+银白+电弧 |

### 4.3 金蝉形象
- 幼虫期：圆润可爱，土黄色，大眼睛
- 每蜕变一次颜色更金，体型更优美
- 最终金蝉：通体金光，半透明翅翼，仙气

## 五、技术映射（改造清单）

### 5.1 直接复用（改名换皮）
- 武器系统 → 虫兵系统（逻辑不变，改名+改图）
- 砖块系统 → 天敌系统（逻辑不变，改外观）

### 5.3 新增
