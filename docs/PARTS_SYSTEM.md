# 飞机芯片系统 · 设计文档

> v5.1 — 2026-03-06
> 参考向僵尸开炮宝石系统
> 每颗芯片 = 1条随机词条 / 5合1升品质 / 洗练重roll
> 纯局外养成

---

## 一、系统概述

飞机有6个部件，每个部件5个芯片槽。
每颗芯片只有**1条随机词条**。
玩家通过抽取、合成升品、洗练追词条来变强。

**核心循环：**
```
抽芯片 → 镶嵌到部位 → 5合1升品质 → 洗练追完美词条 → 变强推更高章节
```

---

## 二、飞机部件

| 部位 | 图标 | 词条池方向 | 解锁条件 |
|------|------|-----------|---------|
| 火控系统 | 🔫 | 攻击/伤害 | 初始 |
| 动力炉 | 🔋 | 元素/属性 | 初始 |
| 装甲模块 | 🛡 | 生存/防御 | 初始 |
| 机动引擎 | 💨 | 移速/闪避 | Ch.15 |
| 战术链路 | 📡 | 收益/辅助 | Ch.30 |
| 改装坞 | 🔩 | 万能(全池) | Ch.50 |

每个部件 **5个芯片槽**，部件解锁即全开。
满配 = 6 × 5 = **30颗芯片**。

---

## 三、芯片结构

每颗芯片只有 **1条随机词条**：

```
┌──────────────────┐
│ 🟣紫 暴击率 +4.9% │
│ 火控系统 · 芯片    │
└──────────────────┘
```

- 词条从该部位的词条池中随机抽取
- 数值在该品质的区间内随机
- 只能装到对应部位（改装坞除外，万能）

---

## 五、5合1升品质

- **5颗同部位同品质** → 合成为高一级品质
- 合成后词条**重新随机**
- 不需要同词条，只要同部位同品质

| 升级 | 消耗 | 累计白色数 |
|------|------|-----------|
| 白→绿 | 5白 | 5 |
| 绿→蓝 | 5绿 | 25 |
| 蓝→紫 | 5蓝 | 125 |
| 紫→橙 | 5紫 | 625 |
| 橙→红 | 5橙 | 3125 |

## 六、洗练

### 机制
- 对已有芯片**重新随机词条**（同部位词条池内重roll）
- 词条类型 + 数值全部重随机
- 洗练后 **二选一**：保留原词条 or 接受新词条（不怕洗废）

### 消耗：焕新试剂
每次洗练消耗 **1瓶焕新试剂**。

### 焕新试剂获取

| 途径 | 数量 | 说明 |
|------|------|------|
| Boss击杀 | 0~1瓶(30%) | 免费但稀少 |
| 章节首通 | 1瓶 | 一次性 |
| 分解橙色芯片 | 2瓶 | 高品质分解附赠 |
| 分解红色芯片 | 5瓶 | |
| **商店购买** | **1瓶=3钻石** | 💰核心付费点 |
| **月卡每日** | **2瓶** | 月卡价值大增 |

### 洗练经济学
- 紫色芯片洗到好词条：平均 3~5 次 = 3~5瓶
- 橙色芯片洗到好词条：平均 5~8 次
- 红色芯片洗到专属词条：平均 10~20 次 = **30~60钻石**
- 这是长期付费驱动力

## 八、获取途径

### 抽奖

**军火商（普通抽）** — 金币
- 单抽：200金币 / 十连：1800金币
- 每日免费1次
- 出白~橙，极低概率红

**黑市（高级抽）** — 钻石
- 单抽：1钻石 / 十连：9钻石
- 红色概率2%
- 保底：50抽必橙，100抽必红
- 怜悯计数永不清零

### Boss掉落
- Boss必掉1颗芯片，首通额外+2
- 不掉红色

| 章节 | 白 | 绿 | 蓝 | 紫 | 橙 |
|------|---|---|---|---|---|
| 1~10 | 50% | 30% | 15% | 5% | 0% |
| 11~30 | 30% | 30% | 25% | 12% | 3% |
| 31~60 | 10% | 25% | 35% | 22% | 8% |
| 61~100 | 0% | 15% | 35% | 35% | 15% |

### 分解
不需要的芯片 → **纳米碎片**
白=1, 绿=5, 蓝=25, 紫=125, 橙=625, 红=3125
碎片在商店兑换指定部位随机芯片包。

## 十、数值公式

```
芯片效果值 = 白色基础值 × 品质倍率(区间内随机)
```

**举例：暴击率芯片 (白色基础值 +1%)**
| 品质 | 数值范围 |
|------|---------|
| ⚪白 | +1.0% |
| 🟢绿 | +1.5%~1.8% |
| 🔵蓝 | +2.0%~2.5% |
| 🟣紫 | +3.0%~3.8% |
| 🟠橙 | +4.5%~5.5% |
| 🔴红 | +6.0%~7.0% |

**设计锚点：**
- 全蓝色芯片 ≈ 够推 Ch.30~40
- 全紫色芯片 ≈ 够推 Ch.50~60
- 橙+红混搭 ≈ Ch.70+

**叠加规则：** 同类词条加法叠加，与永久升级/武器商店为不同乘区。
